# 📊 Bank Marketing Analysis

## Predicting Term Deposit Subscription Using Machine Learning

### 📌 Overview
This project analyzes customer data from a bank's marketing campaigns to identify which customer attributes and behaviors are most predictive of term deposit subscription.

The objective is to help banks improve marketing effectiveness by targeting customers who are more likely to subscribe.



## 🎯 Objective

Build a machine learning model that predicts whether a customer will subscribe to a term deposit based on demographic, financial, and campaign-related information.



## 📂 Dataset

The dataset contains customer and campaign information, including:

- Demographic details (Age, Job, Marital Status, Education)
- Financial information (Balance, Housing Loan, Personal Loan)
- Campaign details (Contact Type, Day, Month, Duration)
- Previous campaign outcomes
- Target Variable: Subscription to a term deposit (Yes/No)

## 🤖 Machine Learning Model

### Random Forest Classifier

The Random Forest algorithm was selected because it:

- Handles both numerical and categorical data effectively
- Reduces overfitting through ensemble learning
- Provides feature importance rankings
- Delivers strong classification performance
