# Bank Marketing Analysis Project

## Objective
Analyze customer behavior and predict term deposit subscription using machine learning.

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Features
- Data Cleaning
- Exploratory Data Analysis
- Visualizations
- Machine Learning Model
- Prediction System

## Machine Learning
Random Forest Classifier was used for prediction.

## Project Workflow
Dataset → Cleaning → EDA → Visualization → ML Model → Prediction
